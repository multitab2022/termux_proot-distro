# termux_proot-distro
termux proot-distro

```
-- Termux 에서 진행합니다.
$ cd ~
$ wget https://github.com/multitab2022/termux_proot-distro/ubuntu2204.sh
$ cp ubuntu2204.sh $PREFIX/etc/proot-distro/ubuntu2204.sh

-- 이후 설치를 진행합니다.
$ proot-distro install ubuntu2204

-- 참고. 설치된 이미지를 지울 때
$ proot-distro remove ubuntu2204
```
